## 🚀 Cara Mulai

### Step 1: Install Dependencies
Buka terminal di folder `protofolio` dan jalankan:
```bash
npm install
```

### Step 2: Jalankan Development Server
```bash
npm start
```

Portfolio akan otomatis membuka di `http://localhost:3000` 🎉

### Step 3: Mulai Customize!
Edit file-file di `src/components/` untuk personalisasi konten Anda.

## 🎯 Langkah Selanjutnya

### 1️⃣ Update Konten (PENTING!)
Edit file di `src/components/`:
- Update nama Anda di Navbar & Hero
- Update bio di About
- Add/edit projects di Projects.js
- Update skills di Skills.js
- Update contact info di Contact.js

### 2️⃣ Customize Design
Edit `tailwind.config.js` untuk:
- Ubah primary color (cyan → warna favorit)
- Ubah secondary color
- Ubah accent color

### 3️⃣ Test Thoroughly
- Run `npm start`
- Test di mobile & desktop
- Test semua buttons & links
- Check console untuk errors

### 4️⃣ Build untuk Production
```bash
npm run build
```

### 5️⃣ Deploy! 🚀
Pilih satu:
- **Vercel** (recommended) - https://vercel.com
- **Netlify** - https://netlify.com
- **GitHub Pages** - Built-in
